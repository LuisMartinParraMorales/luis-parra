This repo is designed to become publishable.

That means the content here should be:

- public-safe

- polished
- evidence-based

Anything that is more private, strategic, or claim-checking heavy should stay in the private/internal repo:

- `/Users/luis.parramorales/.codex/worktrees/3594/day-to-day/career-foundation`
